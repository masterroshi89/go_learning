# Go_Learning

## First few commands after installation

```
go version
go env
go help
go get - downlaod the dependencies , dependency manager
go fmt <file name > - to format the file.
go run <file name> - to run your file.
go build <file name> - create a build of the file.

GOBIN - property to store generated artifacts from go install command.
To set
Environment variables > new variable > name : > Value : path to file

add GOBIN path inside the path variable to run the exe generated.
```
